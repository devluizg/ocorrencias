# Sistema de Ocorrências Escolar

Sistema para registro de ocorrências escolares com acesso via navegador e visualização para coordenação.

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Database:** Supabase (PostgreSQL)
- **Hosting:** GitHub Pages

## Configuração

### 1. Banco de Dados (Supabase)

Crie a tabela no SQL Editor do Supabase:

```sql
CREATE TABLE occurrences (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  student_name TEXT NOT NULL,
  class TEXT NOT NULL,
  type TEXT NOT NULL,
  gravity TEXT NOT NULL,
  description TEXT,
  teacher TEXT NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

ALTER TABLE occurrences ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow public access" ON occurrences FOR ALL USING (true);
```

### 2. Credenciais

1. Copie `js/config.example.js` para `js/config.js`
2. Preencha com suas credenciais do Supabase:
   - **URL:** Project Settings → API → Project URL
   - **Key:** Project Settings → API → New publishable key

### 3. Deploy no GitHub

1. Crie um repositório
2. Faça push dos arquivos (exceto `js/config.js`)
3. Settings → Pages → Branch: main → Save
